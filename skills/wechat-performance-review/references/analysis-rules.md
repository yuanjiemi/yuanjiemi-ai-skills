# 公众号数据复盘规则

## 指标定义

- 总互动数 = 点赞数 + 分享数 + 爱心/在看数。
- 互动率 = 总互动数 ÷ 阅读量。
- 缺失数字留空，不把缺失当 0；接口明确返回 0 时才记录 0。
- “点赞”和“爱心/在看”只有在来源明确为两个不同字段时才分别记录，避免重复计算。

## 内部基准

只用同一账号、相近复盘阶段的数据比较。样本至少 3 篇时计算历史中位数：

- 高于基准：指标高于中位数 20% 以上。
- 接近基准：位于中位数的 80%–120%。
- 低于基准：低于中位数 20% 以上。

样本不足时写“样本不足”，仍可描述数据和内容结构，但不得宣称稳定规律。

## 分析边界

- 标题、选题、开头、发布时间和 CTA 都可能影响表现，不把其中任何一个单独写成确定原因。
- 每条“可能原因”必须同时说明验证方法，例如下一篇只调整标题结构，其他条件尽量保持一致。
- 不同账号粉丝基础不同，不直接比较绝对阅读量。
- 没有新增关注、加微信、预约或成交数据时，不评价转化率和收入效果。
- 不因为阅读量高就复制原文表达；只提炼结构和方法。
