# 微信公众号排版预览标准

仅在用户要求排版、颜色、HTML 预览或公众号可复制版本时读取。

## 默认样式

- 正文字号：17–18px。
- 行高：1.7–1.8。
- 正文颜色：`#222222`。
- 段后距：12–16px。
- 背景：`#FFFFFF`。
- 内容最大宽度：677px。
- 默认使用左对齐；只有品牌样式明确要求时才整篇居中。
- 视频承接短文默认一段一个完整意思，通常 1 句、约 20–55 个汉字；单段超过约 70 个汉字时在语义转折处拆分。
- 排版阶段保留审核稿的标题、文字和段落边界，不重新合并、拆散或改写。
- 不用空的 `<p>` 标签制造间距。

## 强调样式

- 只强调核心对象、关键数字、真实结果和反差结论。
- 强调色默认使用 `#D32F2F`，同时设置 `font-weight: 700`，避免只靠颜色传递信息。
- 每 300 字通常不超过 3–5 处强调。
- 不使用多种高饱和颜色，不整段标红。

推荐写法：

```html
<strong style="font-weight:700;color:#D32F2F;">15分钟</strong>
```

## 视频与 CTA

- 视频插入位使用浅灰虚线框，居中显示，不混入正文段落。
- CTA 前增加细分隔线和足够留白。
- CTA 正文保持原文，不改变领取条件、链接或承诺。
- 默认 CTA 标题使用 22px、加粗、`#171717`。
- 默认 CTA 正文中每次出现的 `《AI 提效模板库》` 都使用 `font-weight:700` 单独加粗。
- 默认 CTA 的扫码引导使用 `#D32F2F`、加粗、居中；箭头居中。
- 默认 CTA 中的 `预约 30 分钟元子 1v1` 使用 `#576B95`、加粗，只强调这段文字。
- 默认 CTA 的 Calendly 链接使用 `#D32F2F` 并加下划线。
- 默认 CTA 其余文字保持 `#222222`，保留原有短段落，不合并。
- CTA 图片使用实际素材时限制为 `max-width:100%; height:auto; display:block; margin:0 auto;`。
- 没有实际图片时使用明确的二维码占位框，不伪造二维码。

## 兼容性

- 使用内联 CSS，不依赖外部样式表、JavaScript、Web 字体或复杂布局。
- 避免 flex、grid、position、动画和交互组件；微信公众号编辑器可能移除这些样式。
- HTML 预览只用于检查视觉层级。复制或通过 API 写入公众号后台后，再检查一次字号、颜色、图片和链接。
